# Dataset

Obtain data from http://ai.stanford.edu/~amaas/data/sentiment/ and unzip into this directory.

Or

```
wget http://ai.stanford.edu/~amaas/data/sentiment/aclImdb_v1.tar.gz
gunzip -c aclImdb_v1.tar.gz | tar xopf -
```
